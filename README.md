## 結構

```
huacheng-crystal/
├── index.html      頁面結構與商品資料
├── styles.css      樣式（米白 + 玫瑰金）
├── script.js       滾動動畫與行動選單
└── favicon.svg     瀏覽器分頁圖示
```

純 HTML / CSS / JS，無需 build，直接由 GitHub Pages 提供。

## 更新內容

```bash
# 1. 改檔案
# 2. 推到 GitHub
git add -A
git commit -m "update: 改了什麼"
git push
```

推送後約 1 分鐘 Pages 會自動重新部署。

### 常見修改

| 想改的東西 | 改哪裡 |
|---|---|
| 換商品圖 | `index.html` 把 `<div class="card-img ...">` 換成 `<img src="...">` |
| 加新商品 | 複製任一張 `.card` 區塊修改 |
| 調色 | `styles.css` 最上面的 `:root` 變數 |
| 改文字 | `index.html` 直接編輯 |

## 在本機預覽

雙擊 `index.html` 即可在瀏覽器打開。
